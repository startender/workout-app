## From chad to Gigachad app
An application that allows you to create your own training concept depending on your preferences, as well as customize the various functional components of a personal training:
- create a workout and add various exercises to it
- creating exercises is possible for different muscle groups
- weight control, number of approaches and training time
- the ability to mark completed workouts
- the ability to manage your own workout list

### Available Scripts

In the project directory, you can run:

### `npm i`
### `npm run dev`
